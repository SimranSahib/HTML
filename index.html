HTML

Hyper Text Markup Language

HTML is a standard markup language used for creating web pages.The full from of HTML is "Hyper text markup language". It is easy to learn and easy to use. defines the structure of the web page the main function of html is to tell the browser that how to display the text written. it is used to assign places to the text means it helps to specify the heading, sub-heading and the parahgraphs.
